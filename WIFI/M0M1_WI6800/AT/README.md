# M0M1系列WIFI模块AT指令(HTTP+MQTT)版本说明

## 发布日期:2019.6.17 版本号:2.3.040

1.增加AT+HTTPLN指令，可以查询HTTP服务器上文件的大小。



## 发布日期:2019.6.13 版本号:2.3.039

1.优化AT指令的结尾判断条件。

1.增加FTP功能，优化内存。

3.修正一些查询错误的问题。

4.解决如果发送数据时出现MQTT断开后重连接慢的问题。
